## 📌 Project Description

This project is an audit of the open-source software **Git**, covering its philosophy, Linux implementation, ecosystem, and comparison with proprietary tools.

Additionally, five shell scripts are developed to demonstrate Linux and open-source concepts.

---

## 🧰 Scripts Overview

### 1. System Identity Report
Displays system details like kernel, user, uptime, and license.

### 2. FOSS Package Inspector
Checks if Git is installed and displays version and description.

### 3. Disk and Permission Auditor
Analyzes system directories for size and permissions.

### 4. Log File Analyzer
Counts occurrences of a keyword in a log file.

### 5. Open Source Manifesto Generator
Generates a personalized open-source statement.
